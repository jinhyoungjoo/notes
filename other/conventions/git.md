# Git Conventions

- This is my personal git convention set for unity among my projects.
- It mainly took inspiration from Conventional Commits.

## Commit Summary Structure
<type>[optional scope]: <description>

- Type (lowercase)
    - feat
    - fix
    - build
    - chore
    - ci
    - docs
    - style
    - refactor
    - perf
    - test
- Description (lowercase)
