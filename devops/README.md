# CI/CD Pipeline

## Stages
- Precommit Checks
- Source
    - Branch Protection
        - ex) Required PR Review
    - Linting
        - ex) GitHub Actions
- Build
    - Build Image & Compile Code
    - Unit Testing
    - Code Coverage Checks
- Test
    - Integration Testing
- Release
    - Ship Image To Registry
